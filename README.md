# joseandgoose.com

Personal site built to demonstrate AI fluency in a product management context. Every feature ships with a Build With Me case study documenting the problem it solves, the AI tools used, and what I'd do differently.

## What's here

- **Writing** — case studies on building with AI, one per week
- **Numerator** — multiplayer number guessing game backed by Supabase
- **Fruit Exchange** — community map for sharing backyard fruit trees
- **Search** — keyword search with live dropdown previews across all content
- **Contact** — form wired to Supabase with email notification via Resend

## Built with

Next.js · TypeScript · Tailwind CSS · Supabase · Vercel
